# Screenshots

Place your dashboard screenshots here. The following files are referenced in the main README:

| File | What to capture |
|---|---|
| `login.png` | The login portal page |
| `overview.png` | The Overview tab with stats and charts |
| `alerts.png` | The Alerts tab (rule alerts + correlated alerts) |
| `monitoring.png` | The Monitoring tab with endpoint risk scores |
| `log-management.png` | The Log Management tab with filters |
| `threat-intel.png` | The Threat Intelligence IP lookup tab |

## Recommended tool

**Linux**: `gnome-screenshot`, `flameshot`, or `scrot`

```bash
# Install flameshot for region-capture screenshots
sudo apt install flameshot
flameshot gui   # select a region and save
```
